## Tic-Tac-Zero

A zero knowledge Neural Network that learns to plays Tic Tac Toe from scratch

### State

* Position of board +  turn
* Position of board:
	row1 + row2 + row3 + turn
	e.g. row: --x
	player 0 or 1
* board:
	--x
	o--
	xox + turn: 0 
	-----> "--xo--xox0"


### Rewards

Win : +1
Draw: 0
Lose: -1
